## 栈

[P5788 【模板】单调栈 - 洛谷 | 计算机科学教育新生态 (luogu.com.cn)](https://www.luogu.com.cn/problem/P5788)

```c++
#include<bits/stdc++.h>
using namespace std;
const int N=3000002;
int si=0;
int n,a[N],s[N],f[N];
int main()
{
	scanf("%d",&n);
	for(int i=1;i<=n;i++)
	{
		scanf("%d",&a[i]);
	}
	for(int i=n;i;i--)
	{
		while(si>0 && a[s[si]]<=a[i]) si--;
		f[i]=si==0? 0: s[si];
		s[++si]= i;
	}
	for(int i=1;i<=n;i++)
	{
		printf("%d ",f[i]);
	}
    return 0;
}
```
